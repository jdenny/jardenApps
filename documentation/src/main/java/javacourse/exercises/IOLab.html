Read Question/Answer pairs from a file

Write a method to open a file, where the fileName is passed as a parameter, and read question/answer pairs from that file.

One approach would be to use a properties file, e.g.:

dog=perro
cat=gato
# note: space is a token separator
good\ morning=buenos dias
See the class java.util.Properties, and in particular its 'load' method.

Alternatively use a text file with prefixes on each line, e.g.

Q: what is the Spanish for dog
A: perro
Q: cat
A: gato
# this is a comment
Q: good morning
A: buenos dias
Use the Reader classes; see demo.io.BasicIO, in particular readTextFromFile.

Sample solution: quiz.QuestionsFromFile

This can be used as the basis for FileQuiz in the Quiz Project: Quiz from File
