Build simple Swing interface
  • run demo.swing.SwingNoEvents: no events & poor layout. Close the application (remains running after closing frame!)
  • run demo.swing.quantum.HelloSwing; events handled; better layout
  • exercise: add another button, to clear input textField
Note, in the actionPerformed, you will need to determine which button has been clicked; there is a method on ActionEvent that can tell you.

Build a Swing interface for Quiz. You should use the following components:
  • a JLabel to present the current question
  • a JTextField to input the answer
  • a set of Radio buttons, for now to show the 2 types of quiz; see demo.RadioSwing for an example
  • a JTextField to show the status: right/wrong, plus running score of rights and wrongs.
