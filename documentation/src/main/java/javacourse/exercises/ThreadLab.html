Run thread.exercise.StopWatchGUI

Press the Go button - be patient with it, as nothing happens for about 8 seconds

Now rewrite the StopWatch class to use threads, so that something happens as soon as you press Go. You shouldn't have to make any changes to the StopWatchGUI class.

There are two ways of making a thread: passing a Runnable to the standard Thread class, or extending the Thread class. We prefer the first approach, but you can try both ways if you want.

Sample solutions:

  • thread.solution.StopWatchGUI (Runnable version)
  • thread.solution.StopWatchGUI2 (Thread version)

Need a bit of help?

Then read on. Note: some text has tooltips

1. Make StopWatch a Runnable

  • 1.1 Define StopWatch so that it implements the Runnable interface.
  • 1.2 Rename go() to run().
  • 1.3 In the StopWatch constructor, create a new Thread, with the stopWatch object as the parameter to the constructor, then start this thread.

2. Make StopWatch a Thread

  • 2.1 Define StopWatch so that it extends Thread.
  • 1.2 Rename go() to run().
  • 2.3 In the StopWatch constructor, start this thread.
