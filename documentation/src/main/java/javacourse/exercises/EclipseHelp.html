Create new Java Project

  • On the Eclipse menu bar, click File, then New, then Java Project; this should open the the dialog: Create a Java Project
  • enter the Project name, e.g. MyProject
  • ensure Project layout is set to:
    Create separate folders for sources and class files.
  • Click Finish

Create new Java Package
  • Within your project, right click on src; from the drop-down menu click New, then Package; this should open the dialog: Create Java Package
  • enter the Package name, e.g. mypackage; use only lower-case letters
  • Click Finish
