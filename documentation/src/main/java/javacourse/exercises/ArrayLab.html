Arrays

Back to Exercise List

  1. Declare and initialise an array of Strings
    print out each String in the array.
  2. Declare an array of integers
    initialise with values 1, 2, 3...
    add together all elements of the array
    print out the sum; for the mathematically-minded, the sum should be n(n+1)/2 where n is the number of elements
  3. Declare an array of chars
    initialise each element to one of the values 'N', 'S', 'E', 'W'
    for each element of the array, print out one of the directions North, South, East or West. E.g. if the array of characters contains 'N', 'E', 'N', 'W', the program will print:
    North
East
North
West
