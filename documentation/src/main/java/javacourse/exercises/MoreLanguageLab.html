VarArgs - methods with variable number of arguments

Define a method that takes a string, and a variable number of ints, and prints out the string plus total from adding the ints. In main method, test this method works with 3 ints, and 1 int.

try calling with no ints; what happens?

add another method with same name, but only one parameter, of type string. Include something in the display that makes it obvious which method is being called. What happens now to method call with no ints? It should invoke the single argument version.

Sample Solution: solution.languageplus.VarArgs

Optional: Update SalesUnit.toString() to use MessageFormat.
