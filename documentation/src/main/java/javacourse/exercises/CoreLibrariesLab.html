Formatting Strings

Define a class to hold the details of a person:

  • name
  • salary
  • friend-count
  • date-of-birth

Define a constructor, and a toString() method, which should produce something like:

name=Don Jenny; salary=£123.40; friends=4,455; DOB=Mon Apr 08 00:00:00 BST 1974
Test the methods.

Rewrite toString() to use the String method:

static String format(String pattern, Object... arguments)

The output should be something like:

name=Don Jenny; salary=£123.40; friends=4,455; DOB=1974-04-08

Sample Solution: solution.CoreLibraries
