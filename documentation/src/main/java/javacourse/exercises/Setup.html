Setup for Java Programming Course

Back to Exercise List

Perform the following steps for each student PC

Install JDK 7 or later

  • browse to http://www.oracle.com
  • select downloads
  • navigate to JDK 7 or later
  • read and accept the license agreement
  • select the download file appropriate to your platform, e.g. jdk-7u5-windows-i586.exe
  • when the file is downloaded, run the executable and follow the instructions

Install Eclipse Java IDE

  • browse to http://www.eclipse.org
  • select downloads
  • select platform (e.g. Windows, Unix)
  • select the appropriate download file, e.g. Windows 32 Bit
  • when the file is downloaded, unzip it into a suitable installation folder e.g. c:\eclipse

Run Eclipse & Install The Course Project

  • open eclipse: navigate to Eclipse installation folder
  • run eclipse.exe
  • File, Import (opens 'Import' dialog), General, Existing Projects into Workspace, (Select root directory), Browse to JavaCourse, (Copy projects into workspace = true), Finish

Optionally provide a network

For the sessions on networks, it is better if the PCs can send each other messages over TCP/IP. This could be a local area network or a wide area network.

Optionally provide a PowerPoint viewer

The slides are included in the course project as PowerPoint slides. It is often useful for the students to view these in their own time. PowerPoint Viewer can be downloaded freely from microsoft.com

Test the installation

Open Eclipse and do the following:
  • From menu bar: File, Import
  • select an import source: General, File System
  • from directory: browse to java course directory
  • select all folders in that directory; click Finish
  • Open JavaCourse, src, first, HelloJava.java
  • right-click on HelloJava.java, and select Run As Java Application
  • check the Console window; it should show "Hello Java World"
  • if the Console window is not open:
    from menu bar: Window, Show View, Console

If a network is provided, check the PCs can PING each other.

If a PowerPoint viewer is available, open one of the PowerPoint files from project folder docs/slides
