Very brief history of Java versions

Back to Exercise List

This is a list of the main features, not included in the early releases of Java, that are taught on this course, showing at which version of Java each one was first released. For a complete list, see the Oracle site, or Java version history on Wikipedia.

java 4
  • language
    • assertions
  • class library
    • Java Web Start
    • non-blocking IO (NIO)
    • JAXP
java 5
  • language
    • generics
    • for each
    • autoboxing/unboxing
    • enums
    • variable args
    • annotations (metadata)
    • static imports
  • class library
    • Formatter
    • Scanner
java 6
  • class library
    • Console class
java 7
  • language
    • binary literals
    • underscores in numeric literals
    • Strings in switch
    • try-with-resources
    • catching multiple exceptions
