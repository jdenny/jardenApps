OBJECT-ORIENTATED PROGRAMMING IN JAVA

ABOUT THE COURSE

This 5-day course offers an intensive introduction to Java, with the goal of enabling participants to understand and utilise the principles of object-orientated programming, so they may use Java to solve real-world problems. Syntax and semantics are just the beginning; the course enables students to appreciate the spirit of Java, in order to obtain the full benefit of the powerful design patterns encouraged by the language.

The course exercises are at two levels: step-by-step instructions to cement the programming concepts just learnt; course projects, where we simply describe the requirements, and allow the student to design and code a solution.

WHAT YOU WILL LEARN

  • Think, design and program in an object-oriented way
  • Develop Java applications to solve real-world problems
  • Build code that is effective, reliable and re-usable
  • Enjoy programming in Java!

COURSE OUTLINE

  • Introduction to the Java development and run-time environment
  • Java syntax
  • Object-Oriented programming
  • Class design
  • Use of interfaces and abstract classes
  • Exception handling
  • JUnit testing
  • Introduction to design patterns
  • Code documentation
  • The Collections framework
  • Core library classes
  • Generics
  • Enums
  • Introduction to the Swing API
  • Annotations
  • I/O
  • Networking
  • Introduction to threading
  • Latest features of Java
  • Preview of Enterprise Java

WHO SHOULD ATTEND

This course is aimed primarily at programmers experienced in another language who need to develop applications in Java.

PREREQUISITES

Participants should already have some experience of programming in another language.
