Getting started with Eclipse

create Eclipse Java Project:

File, New, Java Project. More detail?

create package named 'first':

right-click on src, New, Package. More detail?

create class HelloJava; add main method to output "Hello Java World"

package first;

public class HelloJava {
    public static void main(String[] args) {
        System.out.println("Hello Java World");
    }
}

run code:

right-click on file-name in Package Explorer, Run As, Java Application. In the Console, you should see: Hello Java World

Sample Solution: first.HelloJava

As a further optional step, let's create a jar file from Eclipse, and run it by double-clicking it. For HelloJava this won't be very interesting, because it won't run it within a console, and so we won't see anything happen. More interesting is to run an application that has a graphical user interface; we will use a simple application called quantum.HelloSwing.

  • right-click on src/swing/quantum.HelloSwing.java and select Export from the drop-down menu; this should open the Export wizard
  • Select: Java/JAR file, Next
  • JAR File Specification: set JAR file, e.g. /temp/quantum.HelloSwing.jar, Next
  • JAR Packaging Options: Next
  • JAR Manifest Specification: Use existing manifest from workspace; Manifest file: /JavaCourse/src/swing/manifest.txt, Finish

Now go to your file explorer, find /temp/quantum.HelloSwing.jar, and double-click; this should run the quantum.HelloSwing application.
