SalesPerson
  • getPhotoFile() return the name of a file holding the sales person's photo
  • addSales() increment sales by the value supplied
  • getSales() return the value of sales (the value of the goods this salesman has sold so far)
  • toString() return the name and salesTarget (using the superclass) plus the sales figure and the photo file name

Define subclass SalesTeam

Define a class SalesTeam, as a subclass of SalesUnit.

  • addMember() add a SalesPerson to the set of sales people belonging to this team.
  • getMember() return the SalesPerson held at the position in the array specified by index. If index is out of bounds, return null.
  • getMemberCt() return the number of sales people belonging to this team.
  • getSales() return the total sales figure for the team, calculated by accumulating the sales figure for each member of the team.
  • toString() return the name and salesTarget and total sales figure for the team

In main() of SalesTest, create a SalesTeam object with some team members, and invoke the SalesTeam methods, including getName(), defined in the superclass. Also obtain and print out the members of the team.
