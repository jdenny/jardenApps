Variables

  1. Declare variables
    • Within method main(), declare a number of variables of different types
    • initialise or assign values to these variables
    • print the values of each variable using a statement of the form:
      System.out.println("variable=" + variable)
  2. Change the values
    • For numeric types, perform arithmetic on the variables
    • for string type, reassign the value, then append another string
    • for boolean type, set the value using a conditional operator
    • print out the new values
