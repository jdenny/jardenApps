Message: Failed to find the style corresponding to the id ...
Location: layout file in Graphical view
Solution: set the AppTheme from the drop-down list above where the screen should be

Symptom: No messages showing in LogCat
Solution may be one of the following:

  • wrong filter selected? Check if any messages in "All messages"
  • set focus on device/simulator: DDMS -> Devices
  • unplug and replug USB cable
  • restart Eclipse!

Message: Warning about deprecated ActionBarActivity
Solution: Replace with AppCompatActivity
