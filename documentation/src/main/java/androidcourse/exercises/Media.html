Playing Media - Part 1

Back to Exercise List

To start off, let's play back some audio. Edit ExplorerFragment.showFile() to use MediaPlayer, e.g.

if (fileName.endsWith(".mp3")) {
   Uri uri =  Uri.fromFile(file);
   this.mediaPlayer = MediaPlayer.create(
         getActivity(), uri);
   this.mediaPlayer.start(); 
}

Provide method to stop the media, e.g.

private void stopMedia() {
   if (this.mediaPlayer != null && this.mediaPlayer.isPlaying()) {
      this.mediaPlayer.stop();
   }
}

Call this new method from various places, e.g.

showFile() // user has chosen a new file
showCurrentDirectory() // back key to parent directory
onStop() // user exiting from activity

Run explorer; select an mp3 file. (Suggestion: look in /sdcard/Download/) It should play the sound file.
