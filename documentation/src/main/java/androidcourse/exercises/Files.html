Creating Quizzes from Files

Back to Exercise List

Add new quiz types to your QuizActivity. Use jarden.quiz.PresetQuiz (defined in module jardenlib), which is a subclass of Quiz. It has the following constructors:

/**
 * Build a Quiz from properties, where for each property,
 * name is the question, and value is the answer.
 */
PresetQuiz(Properties properties) throws IOException

/**
 * Build a Quiz from a List of QuestionAnswer objects.
 * QuestionAnswer has a constructor:
 *    public QuestionAnswer(String question, String answer)
 */
public PresetQuiz(List<QuestionAnswer> qaList)

/**
 * Build a Quiz from the InputStream. Assumes the inputStream contains
 * text in the form
 *		Q: question1
 *		A: answer1
 *		Q: question2
 *		A: answer2
 *		etc
 */
public PresetQuiz(InputStreamReader isReader) throws IOException

Retrieve Quiz data from various sources:

  • file; e.g. use res/raw/capitals.properties from sample solution module androidcourse
  • server (HTTP); use URL of https://sites.google.com/site/amazequiz/home/problems/spanish.txt?attredirects=0&d=1
  • database; run module jardenprovider; it has a sample Activity to test if the database is working; link to this provider using jarden.provider.engspa.EngSpaContract in module jardenlib; the key constants you need are:
    EngSpaContract.ENGLISH
EngSpaContract.SPANISH
EngSpaContract.CONTENT_URI_ENGSPA
Use buttons or menu items or a spinner to choose quizType; set answerEditText type to number or text depending on quizType.
