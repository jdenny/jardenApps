You are designing a booking system for a hotel. For a room reservation, what attributes and methods are required?

The hotel wants to extend the booking system to include bookings in the sports club of the hotel (pool, tennis, sauna, massage, gym); design a class outline for the sports booking class.

Redesign the classes to put the shared attributes and methods in a super-class. Which methods would be similar, but have different implementations?

Finally add a restaurant booking class to your design.
